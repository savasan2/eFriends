# eFriends

## サイト概要
あるユーザーが別のユーザーにアプローチをし、そのアプローチが承認されるとメッセージが送れるようになる。

### サイトテーマ
ゲーマー同士のマッチングサイト

### テーマを選んだ理由
気軽に遊べる人を探せる便利なサービスがあったらいいと思ったため

### ターゲットユーザ
オンラインゲームをプレイしている人

### 主な利用シーン
同じゲームで遊びたい友達を探すとき
